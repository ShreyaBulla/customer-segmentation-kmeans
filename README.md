# Customer Segmentation using K-Means

##Project Overview
In this project, I used **K-Means Clustering** to group customers based on their Annual Income and Spending Score. This type of analysis helps businesses create targeted marketing strategies for different customer segments.

## Key Features
* **Data Exploration:** Visualized the relationship between age, income, and spending.
* **The Elbow Method:** Used to find the optimal number of clusters ($k=5$).
* **Cluster Visualization:** Created 2D plots to show the 5 distinct customer groups.

## 🛠️ Technologies Used
* **Python**
* **Pandas** for data manipulation
* **Scikit-Learn** for the K-Means algorithm
* **Matplotlib/Seaborn** for data visualization
